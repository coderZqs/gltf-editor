# glTF 编辑器

#### 该项目是一款简易版的glTF编辑器，拥有上传、编辑模型材质、位移、旋转等与下载模型功能。

## 技术栈

- 前端框架：Vue3 + pinia
- 3D框架：threeJS
- UI框架：ant-design-vue

## 编辑功能支持
+ [x] 位移
+ [x] 旋转
+ [x] 缩放
+ [x] 颜色
+ [x] 粗糙度
+ [x] 金属度
+ [x] 贴图
+ [x] 自发光贴图
+ [x] 透明贴图
+ [x] 凹凸贴图
+ [x] 法线贴图
+ [x] 位移贴图
+ [x] 粗糙贴图
+ [x] 金属贴图
+ [x] 面
+ [x] 混合模式
+ [x] 透明度

## 安装

### 安装依赖
```
npm install
```
### 本地运行
```
npm run dev
```

### 打包
```
npm run build
```
